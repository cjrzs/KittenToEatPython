# 欧拉函数

欧拉函数：数字 1~n 中与 n 互质 的数的 个数 ，记为φ(n)\varphi(n)φ(n)。

互质的数：公约数只有 1 的两个数。

例如：φ(12)=4\varphi(12)=4φ(12)=4，其中 1~12 中共有 4 个与 12 互质的数，分别是 1，5，7，11 。

# 一个重要性质

这里我们依赖欧拉函数的一个重要性质欧拉函数是一个积性函数，也就是当 n 与 m 互质时，公式φ(nm)=φ(n)∗φ(m)\varphi(nm)=\varphi(n)*\varphi(m)φ(nm)=φ(n)∗φ(m) 成立。

    例如：12=22∗312=2^2*312=22∗3，那么φ(12)=φ(22)∗φ(3)\varphi(12)=\varphi(2^2)*\varphi(3)φ(12)=φ(22)∗φ(3)

  2. 此时，问题转换为求φ(pkak)\varphi({p_k}^{a_k})φ(pk​ak​)。

    1. 1~pkak{p_k}^{a_k}pk​ak​ 的中共有pkak{p_k}^{a_k}pk​ak​ 个数字，因为pk{p_k}pk​ 是 质数 ，所以与pkak{p_k}^{a_k}pk​ak​ 不互质 的数就是pk{p_k}pk​ 的倍数 (因为他们的公约数除了 1 之外，一定还有pk{p_k}pk​ 本身)，也就是pk，2pk，3pk，...pkak−1∗pk{p_k}， 2{p_k}， 3{p_k}，...{p_k}^{a_{k-1}}*p_kpk​，2pk​，3pk​，...pk​ak−1​∗pk​，总共有pkak−1{p_k}^{a_{k-1}}pk​ak−1​ 个。
      例如：pk=3,ak=5，此时共有 (2*3,3*3,4*3, ... ,3^4 * 3) 是与 3 不互质的数。

    2. 所以，用总数pkak{p_k}^{a_k}pk​ak​ 去掉其中非互质的个数pkak−1{p_k}^{a_{k-1}}pk​ak−1​，剩下的就是与pkak{p_k}^{a_k}pk​ak​ 互质的。也就是 $\varphi (p_k}({a_k))=p_k}({a_k)-p_k}({a_{k-1)}=p_k}({a_k)*(1-\frac{1}{p_k}) $。

# 筛法求欧拉函数

问题升级：要求出 1~n 中所有数字的欧拉函数。

此时，我们发现如果还用上面的方法求 1~n 中每个数字的欧拉函数，那么每个数都要分解质因数，时间复杂度会来到O(nn)O(n\sqrt{n})O(nn​) 。

那么，回想一下在上一篇文章中，我们 求1~n中的质数 时候是怎么做的呢？

我们使用了一个线性筛法，在O(n)O(n)O(n) 的时间复杂度下就求出来了 1~n 中的质数。

而在求欧拉函数的时候，我们也要分解质因数，因此可以在求解质数的时候顺便把欧拉函数也求出来。

首先复习一下，质数的线性筛是怎么做的。

从代码中可以比较容易的分析到，筛选过程总共分为两部分，一部分是质数部分，另一部分是筛掉的合数部分。

# 质数部分

在质数部分，求质数的欧拉函数是非常简单的。从定义出发很容易想到 质数n 的欧拉函数就是φ(n)=n−1\varphi(n)=n-1φ(n)=n−1。

因为 质数的因数只有1和它本身 ，而欧拉函数的定义是 1~n中与n互质的个数 ，那么除了 n 之外，1~n-1 中所有数都与 n 只有 1 这一个共同因数，所以它们都与 n 互质，总个数就是 n-1 个。

用上小节的求欧拉函数的公式表示就是：φ(n)=n∗(1−1n)=n−1\varphi(n)=n*(1-\frac{1}{n})=n-1φ(n)=n∗(1−n1​)=n−1 。

# 合数部分

  1. 当触发中止条件 i % p[j] == 0 时，说明 pj 是 i 的质因子。

    从上节欧拉函数的推导过程中，我们发现一个数的欧拉函数计算过程和其质因子的指数是没关系的，只与质因子本身有关。

    例如：当 n=6 时候，分解质因数结果是2∗32*32∗3，那么求欧拉值：φ(6)=6∗(1−12)∗(1−13)=2\varphi(6)=6*(1-\frac{1}{2})*(1-\frac{1}{3})=2φ(6)=6∗(1−21​)∗(1−31​)=2。
    当 n=12 时候，分解质因数结果是22∗32^2*322∗3，因此，它求欧拉值是φ(12)=12∗(1−12)∗(1−13)=4\varphi(12)=12*(1-\frac{1}{2})*(1-\frac{1}{3})=4φ(12)=12∗(1−21​)∗(1−31​)=4。

    举这个例子，是想说明上面的结论。即，求欧拉值的过程中， 6 和 12 都有相同的质因数 2 和 3 。虽然 12 的 质因子2 的指数是 2 ，而 6 的 质因子2 的指数是 1 ，但这并没有影响(1−12)∗(1−13)(1-\frac{1}{2})*(1-\frac{1}{3})(1−21​)∗(1−31​) 这部分。

    所以，当φ(i∗pj)\varphi(i*p_j)φ(i∗pj​) 中，如果pjp_jpj​ 是 i 的质因子之一，那么φ(pj)=pj∗(1−1pj)\varphi(p_j)=p_j * (1-\frac{1}{p_j})φ(pj​)=pj​∗(1−pj​1​) 中的(1−1pj)(1-\frac{1}{p_j})(1−pj​1​) 已经在φ(i)\varphi(i)φ(i) 的推导过程中被计算过了，所以，我们得到最终公式：φ(i∗pj)=pj∗i∗(1−1p1)∗...∗(1−1pj)∗...∗(1−1pk)=pj∗φ(i)\varphi(i*p_j)=p_j *i *(1-\frac{1}{p_1}) *...*(1-\frac{1}{p_j}) *...*(1-\frac{1}{p_k})=p_j * \varphi(i)φ(i∗pj​)=pj​∗i∗(1−p1​1​)∗...∗(1−pj​1​)∗...∗(1−pk​1​)=pj​∗φ(i)。

  2. 当不满足终止条件，也就是 i % pj != 0 ，此时pjp_jpj​ 和 i 是互质的，那么根据积性函数的性质，可以得到φ(pj∗i)=φ(pj)∗φ(i)\varphi(p_j*i)=\varphi(p_j)*\varphi(i)φ(pj​∗i)=φ(pj​)∗φ(i)。 因为pjp_jpj​ 是质数，所以φ(pj)=pj−1\varphi(p_j)=p_j-1φ(pj​)=pj​−1。最后，得到最终公式：φ(pj∗i)=(pj−1)∗φ(i)\varphi(p_j*i)=(p_j-1)*\varphi(i)φ(pj​∗i)=(pj​−1)∗φ(i)。
